## 📂 File Structure Overview

After extraction, the folder contains:

- `/templates/` — The main folder with 50+ prompt files organized by use case (e.g., Emails, Marketing, Brainstorming)
- `README.md` — This file explaining how to use the templates
- `LICENSE` — Details on how the templates can be reused (usually open source or free use)

Each template file uses simple language to guide the AI. Most follow the RCTFE framework:

- **Role:** Defines who the AI should act as
- **Context:** Provides background information
- **Task:** States what the AI should do
- **Format:** Specifies how the output should appear
- **Example:** Shows example input/output for clarity
